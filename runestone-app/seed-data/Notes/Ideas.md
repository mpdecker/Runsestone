# Project Ideas

A collection of ideas for Runestone features and use cases.

## Feature Ideas

- [ ] **Browser extension** for one-click web clipping
- [ ] **Mobile quick capture** with voice notes
- [ ] **PDF annotation** with automatic extraction
- [ ] **Calendar integration** for daily notes
- [ ] **Publishing** selected notes as a website
- [ ] **Collaborative editing** via remote server mode

## Use Cases

### Research
Using [[Knowledge Graphs]] and [[Semantic Search]] to map research papers:
1. Import PDFs via document import
2. Extract entities and concepts via [[Ollama]]
3. Explore connections in the graph view
4. Chat with the graph to synthesize findings

### Project Management
1. Create notes for tasks and milestones
2. Link deliverables to requirements
3. Use tags for priority and status
4. Daily notes for progress tracking

### Learning
1. Take notes while reading
2. Use `[[wiki links]]` to connect concepts
3. Review the [[Graph Theory]] behind it all
4. Spaced repetition via random note

## Dependencies

Many of these ideas depend on [[Vector Databases]] and the [[Architecture]] being in place.

> [!info] Contribute
> If you're interested in implementing any of these, check out the plugin system in `src/lib/plugin-manager.ts`.
